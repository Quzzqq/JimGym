# JimGym

Веб-приложение сайта спортзала JimGym. React 18 + TypeScript + Tailwind CSS.

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:5173

## Деплой на GitHub Pages

```bash
npm run deploy
```

Предварительно в репозитории включите GitHub Pages (Settings → Pages → Source: gh-pages branch).
